# Smart Voice Based Email System for Visually Impaired

## Project Description
The Smart Voice Based Email System is designed to help visually impaired users send, receive, and manage emails using voice commands. The system eliminates the need for a traditional graphical interface and allows users to interact through speech.

## Objectives
- To provide an accessible email system for visually impaired users
- To enable voice-based interaction for sending and reading emails
- To simplify email communication using speech recognition and text-to-speech

## Features
- Voice-based email sending
- Reading received emails using audio output
- Speech-to-text conversion for composing emails
- Text-to-speech for reading emails
- Simple and user-friendly interface

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3 (Text-to-Speech)
- smtplib (SMTP protocol)
- imaplib (IMAP protocol)

## System Requirements
- Python 3.x
- Internet connection
- Microphone (for voice input)
- Speaker (for audio output)

## Installation
1. Clone the repository:
   git clone https://github.com/sakshi21shinde/Smart-voice-based-email-system-for-Visually-Impaired.git

2. Navigate to the project folder:
   cd Smart-voice-based-email-system-for-Visually-Impaired

3. Install required libraries:
   pip install -r requirements.txt

## Usage
1. Run the main Python file:
   python email_system.py

2. Follow voice instructions to:
   - Send emails
   - Read inbox messages
   - Perform email-related actions

## Project Structure
- email_system.py        : Main application file
- face_login.py          : Face authentication module
- registration_new.py    : User registration module
- train.py               : Model training script
- user_data.db           : Database file
- haarcascade_frontalface_default.xml : Face detection file
- Image/                 : Image dataset
- voice files (.mp3)     : Audio responses

This project demonstrates how voice technology can be used to improve accessibility and provide independence to visually impaired users in digital communication.
